
# Backend

## Setup

1. Create virtualenv:
   ```bash
   python -m venv venv
   source venv/bin/activate  # or `.env\Scripts\activate` on Windows
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Copy `.env.example` to `.env` and fill in real values.
4. Run the server:
   ```bash
   python app.py
   ```
